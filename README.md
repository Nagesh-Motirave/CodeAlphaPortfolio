🌐 Personal Portfolio Website

📌 Overview

This is a **responsive personal portfolio website** built using **HTML** and **Tailwind CSS**.
It showcases my skills, projects, and contact information in a clean and modern UI.

The goal of this project is to create a professional online presence and demonstrate front-end development skills.

---

🚀 Features

* Responsive design (Mobile, Tablet, Desktop)
* Modern UI using Tailwind CSS
* Smooth navigation between sections
* Project showcase section
* About Me section
* Contact section
* Clean and minimal layout

---

🛠️ Technologies Used

* HTML5
* Tailwind CSS
* JavaScript (optional for interactions)

---

   ```

   ```
